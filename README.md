# 疫情防治大作戰！

可直接點擊觀看 [遊戲遊玩介紹影片](https://youtu.be/iicNh5AgdOk)

### 遊戲規則
玩家共有三條命，當血量被扣除歸零時，則自動減少一條生命。<br>
可藉由**發射子彈**消滅病毒、並獲得積分。<br>
若遊戲結束時積分大於5000分，則會顯示遊戲勝利畫面。<br>

### 道具功能
由遊戲畫面上方隨機掉落以下幾種道具，若碰到角色即會觸發對應功能。
* **冠狀病毒**－扣除玩家血量
* **口罩**－觸發補血功能
* **疫苗**💉－啟動5秒鐘的子彈加成效果

### 玩家操作說明
* 以 `↑` `⭣` `←` `→` 移動玩家位置
* 按 `(space)` 空白鍵可發射子彈
